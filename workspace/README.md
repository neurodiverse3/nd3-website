# Workspace Folder

This folder holds local-only material that supports the website project but is not part of the deployed site.

## Subfolders

- `content/` for product PDFs, packaged deliverables, writing drafts, and template source files
- `media/` for design assets and preview videos
- `artifacts/` for audits, screenshots, temporary exports, and other generated outputs

The live website still runs from the repository root using `src/`, `public/`, and the root configuration files.
